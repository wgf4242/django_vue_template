Django + Vue3 Project Template

# Get started

**Client install**

```
cd client
pnpm install
```
Backend install
```
cd backend
pip install -r requirements.txt
```

**Config Database**

set Your Database in settings.py
```
# mysql
cp backend/django_vue_template/my.cnf.bak backend/django_vue_template/my.cnf
```

**Start proejct**

```
# client
cd client
pnpm dev

# backend
cd backend
python3 manage.py runserver
```
